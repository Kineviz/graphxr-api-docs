  • applyLayout(__namedParameters: LayoutResult, tweenOverrides?: TweenLayoutOverrides): Promise<NodeToPosition>
    • Defined in applyLayout.ts:31

    Sets the positions of Nodes. Optionally animate the position. If tween provided with the LayoutResult, use it. Override the default tweens returned by (for example) line() and circle() by providing TweenLayoutOverrides. If tween is undefined, set all the positions immediately

    since

    0.0.1

    example
    GraphXR.applyLayout(GraphXR.line())
GraphXR.applyLayout(GraphXR.circle(), {duration: 3000})
GraphXR.applyLayout(GraphXR.cube(), {duration: 3000, interpolate: myInterpolate})
GraphXR.applyLayout(
  {
    final: new Map(GraphXR.getNodes().map(n => [n, GraphXR.makeRandomPosition()])),
    tween: {
      duration: 3000,
      interpolate: myInterpolate,
    }
  }
)

    Parameters

    • Optional tweenOverrides: TweenLayoutOverrides
